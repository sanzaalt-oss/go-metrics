# go-metrics

A lightweight metrics library for Go applications providing counters, gauges, histograms, meters, and timers.

> **Note:** This library is primarily developed for internal use by Sanzaal Technology Pvt. Ltd. Features and development priorities are aligned with our internal use cases. We welcome the community to use and contribute, though please note that feature requests may be evaluated based on our internal requirements.

## Installation

```sh
go get -u github.com/sanzaalt-oss/go-metrics
```
